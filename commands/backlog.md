---
description: Backlog のプロジェクト管理操作（課題検索・作成・更新・コメント）
---

Backlog MCP サーバーを使って、指定された操作を実行してください。

## 操作の分類

### 課題の参照

- `課題一覧` — プロジェクトの課題一覧を表示する
- `課題詳細 PROJ-123` — 指定した課題の詳細を表示する
- `担当課題` — 自分が担当している課題を表示する
- `未完了課題` — 未対応・処理中の課題を表示する

### 課題の作成・更新

- `課題作成` — 新しい課題を対話形式で作成する
- `ステータス更新 PROJ-123 処理中` — 課題のステータスを変更する
- `コメント追加 PROJ-123 コメント内容` — 課題にコメントを追加する
- `担当変更 PROJ-123 田中太郎` — 担当者を変更する

### プロジェクト管理

- `マイルストーン一覧` — マイルストーンと進捗を表示する
- `バーンダウン` — マイルストーンのバーンダウンチャートを表示する

## 出力フォーマット

課題一覧は以下の形式で出力してください：

```
## Backlog 課題一覧

### 処理中
- [PROJ-XXX] 件名（担当者） 期限: YYYY/MM/DD

### 未対応
- [PROJ-XXX] 件名（担当者） 期限: YYYY/MM/DD
```

## 使用例

```
/backlog 今週の担当課題を表示して
/backlog PROJ-123 の詳細を見せて
/backlog ログインバグの課題を作成して、優先度は高、期限は来週金曜日
/backlog PROJ-123 に「修正完了、テスト待ち」とコメントして
```
