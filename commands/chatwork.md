Chatwork MCP サーバーを使って、指定された操作を実行してください。

## 操作の分類

### メッセージ操作

- `メッセージ一覧 [ルーム名]` — 指定ルームの最新メッセージを表示する
- `送信 [ルーム名] [メッセージ]` — メッセージを送信する
- `日報送信 [ルーム名]` — 日報テンプレートで送信する
- `障害報告 [ルーム名]` — 障害報告テンプレートで送信する

### ルーム管理

- `ルーム一覧` — 参加しているルームを一覧表示する
- `ルーム詳細 [ルーム名]` — ルームの情報とメンバーを表示する

### タスク管理

- `タスク追加 [ルーム名] [タスク内容] [期限]` — タスクを追加する
- `タスク一覧 [ルーム名]` — 未完了タスクを表示する

## 出力フォーマット

メッセージを送信する前に確認を求めてください：

```
以下の内容を「{ルーム名}」に送信します：

{メッセージ内容}

送信してよいですか？（はい/いいえ）
```

## 使用例

```
/chatwork ルーム一覧を表示して
/chatwork 開発チームルームに「PROJ-123 修正完了しました」と送信して
/chatwork 開発チームルームに今日の日報を送って
/chatwork 開発チームルームに「DB メンテナンス」のタスクを期限4/15で追加して
```
