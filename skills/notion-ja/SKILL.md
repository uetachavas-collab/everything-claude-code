# Notion 日本語操作スキル

Notion を日本語環境で効果的に活用するためのワークフロー。

## 使いどき

- 会議の議事録を Notion に作成するとき
- タスク・プロジェクト管理データベースを操作するとき
- 社内 Wiki を更新・参照するとき
- ドキュメントを検索・作成するとき
- データベースのレコードを追加・更新するとき

## セットアップ

### MCP サーバーの設定

`~/.claude.json` の `mcpServers` に以下を追加する：

```json
"notion": {
  "command": "npx",
  "args": ["-y", "@notionhq/notion-mcp-server"],
  "env": {
    "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer あなたのトークン\", \"Notion-Version\": \"2022-06-28\"}"
  }
}
```

### インテグレーショントークンの取得

1. Notion にログインし、[notion.so/my-integrations](https://notion.so/my-integrations) にアクセスする
2. 「新しいインテグレーション」を作成する
3. トークンをコピーする
4. 連携したいページ・データベースでインテグレーションを「接続」する

## 仕組み

### 議事録の作成

```markdown
# 議事録

**日時**: 2026年04月10日（木）14:00〜15:00
**場所**: 会議室A / Zoom
**参加者**: 山田太郎、田中花子、鈴木一郎
**議事録担当**: 山田太郎

---

## アジェンダ

1. 先週の進捗報告
2. 今週の作業計画
3. 課題・相談事項

---

## 議事内容

### 1. 先週の進捗報告

- 山田: ログイン画面のバグ修正完了（PROJ-123）
- 田中: 検索API調査中、来週完了予定

### 2. 今週の作業計画

...

---

## アクションアイテム

| タスク | 担当 | 期限 |
|---|---|---|
| APIドキュメント更新 | 山田 | 4/15 |
| テスト環境の再構築 | 田中 | 4/17 |

---

## 次回

**日時**: 2026年04月17日（木）14:00〜
```

### データベース操作

日本語のプロパティ名を持つデータベースを操作できる。

```
# よく使うプロパティ名（日本語対応）
担当者 → Assignee
期限 → Due Date
ステータス → Status
優先度 → Priority
カテゴリー → Category
```

### ステータス管理

```
未着手 → 進行中 → レビュー中 → 完了
                ↓
              保留中
```

## 使用例

```
今日の14時から15時の開発チームMTGの議事録を Notion に作成してください。
参加者は山田・田中・鈴木の3名です。

Notion のタスクデータベースで「山田太郎」が担当、ステータスが「進行中」の
タスクを一覧表示してください。

「インボイス制度」に関する Notion ページを検索してください。
```
