# コマンドクイックリファレンス（日本語版）

> スラッシュコマンドで Claude Code セッションから呼び出せます。

---

## 基本ワークフロー

| コマンド | 機能 |
|---|---|
| `/plan` | 要件整理・リスク評価・実装計画の作成（コード変更前に確認を求めます） |
| `/learn` | セッションから学習パターンを抽出して再利用可能なスキルに変換 |
| `/aside` | メイン作業を中断せずにサイドの質問をする |
| `/docs` | ライブラリ・フレームワークのドキュメントを参照する |

---

## プロジェクト管理ツール連携

| コマンド | 機能 |
|---|---|
| `/backlog` | Backlog の課題検索・作成・更新・コメント追加 |
| `/chatwork` | Chatwork のメッセージ送信・タスク追加・ルーム管理 |
| `/jira` | Jira チケットの操作（検索・作成・ステータス更新） |
| `/notion` | Notion のページ・データベース操作（議事録作成等） |

---

## コンテンツ・調査

| コマンド | 機能 |
|---|---|
| `/plan` | 実装計画の作成 |
| `/update-docs` | プロジェクトのドキュメントを更新 |
